dotfiles
========

Here is a repository for my dotfiles.

Vim
---

To use my Vim/Neovim configuration:

    cd vimfiles
    sh usevimconfig.sh

tmux
----

To use my tmux configuration:

    cd tmuxfiles
    cp .tmux.conf $HOME

This configuration file is compatible with tmux 3.4.

Git
---

To use my Git configuration:

    cd gitfiles
    cp .gitconfig $HOME
